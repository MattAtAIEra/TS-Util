Events Mediator Pattern

The Events bus is the central nervous system. All modules communicate through typed events.

Code
// Subscribe to lifecycle events
import { Events } from 'ts-util-core';

Events.on('ajax:before', ({ url }) => {
  console.log(`Loading ${url}...`);
});

Events.on('ajax:after', ({ url }) => {
  console.log(`Done: ${url}`);
});

Events.on('validation:invalid', ({ labelNames }) => {
  console.log(`Missing: ${labelNames}`);
});

AJAX Facade + Template Method

AJAX.request() orchestrates validate → emit → fetch → emit. AJAX.requestJSON() adds automatic JSON parsing.

Code
// Simple POST request
await AJAX.request({
  url: '/api/users',
  data: { name: 'Alice' },
  success: (res) => log('Response received'),
  noblock: true,
});

// POST with form auto-serialization
await AJAX.request({
  url: '/api/save',
  form: document.getElementById('myForm'),
});

// JSON with typed response
const user = await AJAX.requestJSON<User>({
  url: '/api/user/1',
});

Validation Strategy + Decorator + Registry

Declare constraints via HTML attributes. The library auto-attaches behavior. Combine multiple constraints with spaces: constraint="required number"

Interactive Form — Try submitting with empty fields
constraint="required number" — only digits, dot, minus allowed
constraint="date" — validates yyyy-mm-dd format on change
constraint="time" — auto-formats on blur (try typing "1430")
constraint="required upperCase onlyEn" — three constraints combined!
Textarea with maxlength="30" — validated on submit
Custom Constraint — extend the library
// Register a custom 'email' constraint
Validation.addConstraint({
  name: 'email',
  attach(el) {
    el.addEventListener('change', () => {
      if (el.value && !el.value.includes('@')) {
        el.value = '';
        alert('Invalid email');
      }
    });
  },
});

// Now use it in HTML:
// <input constraint="required email" labelName="Email" />

Formatting Registry Pattern

Declare format="..." on inputs. The library auto-applies input masks. Type freely — the mask formats as you go.

Built-in Formatters
format="idNumber" — [A-Z] + 9 digits
format="date" — auto-inserts dashes
format="time" — auto-inserts colon
Custom Formatter — add your own
// Register a phone formatter
Formatter.add({
  key: 'phone',
  format: (el) => {
    el.placeholder = '09XX-XXX-XXX';
    el.addEventListener('input', () => {
      let v = el.value.replace(/\D/g, '');
      if (v.length > 4) v = v.slice(0,4)+'-'+v.slice(4);
      if (v.length > 8) v = v.slice(0,8)+'-'+v.slice(8);
      el.value = v.slice(0, 12);
    });
  },
});

MSG Facade Pattern

Vanilla DOM dialogs — no 3rd-party library needed. Fully styleable via CSS classes.

Code
// Auto-closing notification
MSG.info('Saved successfully!', {
  autoclose: 3000,
  title: 'Success',
});

// Modal that must be dismissed
MSG.modal('Session expired.', {
  title: 'Warning',
});

// Confirmation dialog
MSG.confirm(
  'Delete Item',
  'Are you sure?',
  () => { /* delete logic */ },
);

VIEW Observer Pattern

Load HTML fragments dynamically. All beforeLoad hooks run automatically — new content gets validation, formatting, everything.

Code
// Register your own beforeLoad hook
VIEW.addBeforeLoad((context) => {
  // Runs on every new DOM fragment
  context.querySelectorAll('.highlight')
    .forEach(el => el.style.background = '#fef08a');
});

// Load HTML into a container
await VIEW.load(targetElement, {
  url: '/api/partial',
});

// Or manually inject + init
targetElement.innerHTML = htmlString;
VIEW.invokeBeforeLoad(targetElement);

Utilities Pure Functions

Standalone helpers — sprintf, formToJSON, isDateValid. No side effects, fully testable.

sprintf
sprintf('Hello %s, you are %d', 'Alice', 30)
// → "Hello Alice, you are 30"

sprintf('Price: $%0.2f', 9.5)
// → "Price: $9.50"

sprintf('Hex: %04X', 255)
// → "Hex: 00FF"
formToJSON + isDateValid
import { formToJSON, isDateValid } from 'ts-util-core';

const data = formToJSON(formElement);
// → { username: "alice", role: "viewer" }

isDateValid('2026-02-24')  // → true
isDateValid('not-a-date') // → false
